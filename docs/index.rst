Welcome to AIODATA
==================

A lightweight ``PostgREST`` proxy that does not get in your way.

Requirements
  .. code-block:: bash

    $ brew install postgresql postgrest

Installation
  .. code-block:: bash

    $ pip3 install aiodata

Check out the :ref:`Quickstart` guide to jump right in.
